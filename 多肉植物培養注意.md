要他們死掉方法許多種
```
```
1. 每天澆水。（它們大多生長在艱苦環境中，不用過度溺愛）
2. 日正當中澆水。（水會馬上變熱然後盆裡會像個蒸氣室把根悶爛）
3. 盆底沒有洞。（因為根部需要空氣流通，能加快水份流通速度）
4. 夏天。（尤其台灣夏天.非常悶熱及潮濕，我自己夏天都清出好多空盆）
5. 蟲害。（最可怕為粉介殼蟲，藥劑時常無法殺死它，身上粉會將藥劑帶走，試著用針一隻隻刺死）
6. 細菌。（常發生生病果凍葉掉在表土開始發爛，並開始傳染到整株死亡）
```

需要注意
```
```
1. 長時間日照，多肉生長在類似沙漠環境，陽光不夠強會發現每片葉子距離開始拉長，這時不華麗也不可愛了。
2. 按品種澆水，水量認真起來會研究不完，以我家環境（全日照、遮雨棚、五樓通風處），秋冬春日夜溫差大，靠近冬天我會選擇 1-2 星期澆水，看情況澆透 ; 夏天、春秋偏熱時少澆水，大概 1 個月 1 次水少量不澆透，有些品種容易乾，需要偶爾噴水霧或移至遮陽區域。
3. 介質最關鍵，想把多肉養好一定要認真探討這部分，環境、品種、澆水習慣都有其配方。我的懶人配法，底層使用大顆粒煤焦石，中間會用中顆粒赤玉土混中細顆粒煤焦石，表土蓋上中粒的赤玉土。每家多肉名店都會賣特調介質配方，依舊不適合環境因素，建議買調配好介質當作中層，底層搭配超疏水介質，比較能預防手癢把土澆透才不至於使根部爛掉，表土最好鋪個單一色赤玉石，簡單好看
```

不同品種的照顧方式
```
```
1. 仙人掌部分夏天放在有洞盆器，盡情澆到透為止，因為生長季所以很需要養分。
2. 常見景天科要小心悶熱、注意日照是否充足、水份過多、低日照會讓它們脖子抽長變得有些醜陋。涼爽季節不過度澆少量水，日夜溫差大的秋季限制水份，某些品種會看到顏色轉紅加密集葉子，就是所謂多肉人在涼爽生長季追求漂亮模樣。（常看到日本、歐洲那些乾冷氣候區域，多肉們大多顏色鮮紅且飽滿）
3. 百合科及菊科孩子挺喜歡喝水，夏天用噴水器添加水霧，不然容易收空盆。
4. 大戟科要特別注意，不是不好養，它們傷口會有白色乳汁，只要碰過記得一定要徹底洗手或乾脆戴手套，有一次在處理時，手指沾到乳汁還洗三次手，但揉眼睛後，馬上開始眼睛灼熱狂流眼淚，到急診處理完才停止，期間眼睛發紅劇痛，若直接濆進眼球，有可能直接瞎掉。
5. 想在室內放植物，請愛用空氣鳳梨，冷氣房內也活很好，只要有些乾泡水15分鐘馬上硬挺，切記泡水後要迅速甩乾，免得會爛根。
